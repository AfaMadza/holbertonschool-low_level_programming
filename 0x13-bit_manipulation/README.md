# Bit Manipulation
At the end of this project, students should be able to:

* Manipulate bits and use bitwise operators

## Environment

* __OS:__ Ubuntu 14.04 LTS
* __Compiler:__ gcc 4.8.4 (C90)
* __Language:__ C

### Primary Concepts

* ``&``, ``^``, ``|``, ``~``
